## Come aggiornare questa baseline

Al termine di ogni ondata, **non** si modifica questo file. Si crea un nuovo file `baseline-YYYY-MM-DD.md` con i nuovi valori, e si confronta il delta contro questa baseline originale.

Il file viene modificato **una sola volta**: quando il freeze termina, per archiviare lo stato finale. A quel punto verrà aggiunta una sezione "Final delta" in fondo.
